Task 28 - ER Diagram Generator (FK Relationships)

How to run:
  ./gradlew :core:test --tests com.dbeagle.er.ERDiagramGeneratorTest

What this validates:
  - ERDiagramGenerator.generate(schemaMetadata, schema) produces:
    - Nodes for 5 tables in the requested schema (sorted by table name)
    - Edges for 3 foreign keys where BOTH endpoints are within that schema
  - Foreign keys that reference a table outside the requested schema are excluded
